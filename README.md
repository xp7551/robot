This is a web interface to control a raspberry pi robot.

**Hardware**
raspberry pi 3B
L298M motor controller
wheeled robot body
pi camera
2600 milli amp phone recharger
ABS project box

**Software to install**
sudo install virtualenv
pip install flask
pip install RPi.GPIO
pip install gpiozero
apache httpd

***Requirements***
Python 2.7
